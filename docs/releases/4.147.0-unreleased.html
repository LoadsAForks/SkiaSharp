Version 4.147.0

Upcoming release · In development · Not yet available on NuGet

Highlights

This release brings an expanded interactive Gallery with 5 new Blazor samples and a redesigned home page, plus significant UX improvements to SkiaFiddle from community contributor @ramezgerges. Apple platform TFM handling is corrected so library projects and app projects receive the right target frameworks. The Uno sample ecosystem is also kept up to date, courtesy of @ramezgerges.

Breaking Changes

None in this release.

New Features

Samples & Gallery

  • 5 new interactive Gallery samples — Adds five new hands-on demos to the Blazor Gallery and redesigns the home page for a better first-run experience. (#3757)
  • SkiaFiddle UX improvements — Numerous usability enhancements to the SkiaFiddle playground. ❤️ @ramezgerges (#3790)
  • Uno sample updated to Uno.Sdk 6.6.0 — Keeps the Basic Uno sample on the latest Uno SDK pre-release. ❤️ @ramezgerges (#3821)

Bug Fixes

  • Apple platform TFM correction — Libraries now use versioned TFMs (e.g. net9.0-ios26.0) while app-head projects use the unversioned form, matching Apple's requirements for the upcoming OS releases. (#3798)
  • Uno Gallery Skottie package downgrade — Fixes a NU1605 package downgrade warning in the Skottie sample inside the Uno gallery. ❤️ @ramezgerges (#3785)
