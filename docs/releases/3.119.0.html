Version 3.119.0

Major milestone release · Released April 29, 2025 · NuGet · GitHub Release

Highlights

SkiaSharp 3.119.0 is the first release built on Skia milestone 119, bringing the latest upstream rendering improvements. This release dramatically expands platform reach with new Direct3D support on Windows, GL on Windows ARM, tvOS Metal views, and first-ever RISC-V 64 and LoongArch64 Linux builds — plus Alpine Linux support. Six community contributors drove the majority of platform and API additions.

Breaking Changes

None in this release.

New Features

Engine

  • Bump Skia to milestone 119 — Upgrades the underlying Skia graphics engine to Chrome milestone 119, bringing rendering improvements and new internal capabilities. (#3062)

GPU & Rendering

  • Direct3D backend on Windows — Compiles Skia with Direct3D support on Windows, enabling D3D-based GPU rendering. ❤️ @Kation (#2823)
  • GL on Windows ARM — Enables OpenGL rendering on Windows ARM64 devices. (#3189)

API Surface

  • Missing SKColorFilter types — Adds previously missing color filter types to the C# API surface. (#2882)
  • SkCanvas::SaveLayerRec — Exposes the SaveLayerRec API for advanced canvas save-layer control. ❤️ @ahmed605 (#2962)

Platform

  • tvOS Metal support — Adds SKMetalView support on tvOS, completing Apple platform GPU coverage. ❤️ @MartinZikmund (#3114)
  • RISC-V 64 Linux builds — Adds native build support for the riscv64 architecture. ❤️ @kasperk81 (#3192)
  • LoongArch64 Linux builds — Adds native build support for the LoongArch64 architecture. ❤️ @4Darmygeometry (#3198)
  • Alpine Linux support — Extends the clang-cross toolchain to support Alpine (musl libc). ❤️ @kasperk81 (#3200)

Security

  • Android NDK updated to r27c — Updates to NDK r27c with 16KB page alignment for modern Android devices. (#3096)
  • Address CVE-2024-30105 — Security fix in build dependencies. ❤️ @pjcollins (#3166)
  • Update libpng to 1.6.44 — Bumps libpng for security and stability improvements. (#3059)

Bug Fixes

  • Fix SKImage.FromPicture implementation — Corrects the image-from-picture creation path. (#3231)
  • Fix incorrect SafeRef call — Fixes an incorrect method call in the safe reference wrapper. ❤️ @kkwpsv (#3143)
  • Fix iOS Simulator Metal performance — Resolves a performance regression when using Metal in the iOS Simulator. ❤️ @taublast (#3156)
  • Fix .NET Framework/Mono packages.config regression — Fixes a regression that broke projects using .NET Framework or Mono with packages.config. ❤️ @sungaila (#3112)
  • Increase Linux support — Broadens compatibility across Linux distributions. (#3209)

Plus several CI and build system improvements.
